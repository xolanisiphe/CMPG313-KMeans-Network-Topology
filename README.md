# K-Means Clustering of Dense Network Topology

## Project Overview
[cite_start]This project explores the application of **K-Means clustering** on a dense network consisting of over 600 interconnected nodes[cite: 5, 13]. [cite_start]Each node represents a real-world IoT or wireless sensor environment, featuring specific spatial coordinates and energy levels[cite: 6].

[cite_start]The goal is to move beyond visual assumptions of dense data and rely on data-driven analysis using **Python (Pandas, NumPy)** to group nodes and analyze energy distribution[cite: 9, 10, 11].

## Learning Outcomes
- [cite_start]Applied K-Means clustering using the predefined `KMeans()` function[cite: 13].
- [cite_start]Analyzed a dense topology with 600+ nodes[cite: 13].
- [cite_start]Interpreted results through **2D network graphs** and **3D surface plots**[cite: 15].
- [cite_start]Evaluated energy distribution and identified critical nodes with the lowest energy[cite: 10, 16].

## Methodology
1. [cite_start]**Clustering:** Nodes were grouped into clusters (testing $k=7$ and $k=3$) based on spatial positioning[cite: 7, 28].
2. [cite_start]**Energy Analysis:** Evaluated how energy is distributed across different clusters to identify network vulnerabilities[cite: 10].
3. [cite_start]**Visualization:** Generated 2D graphs to see cluster boundaries and 3D surface plots for complex spatial analysis[cite: 10, 15].
